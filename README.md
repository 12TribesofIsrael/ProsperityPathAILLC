## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/12TribesofIsrael/ProsperityPathAILLC.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```
